Effet de Serre - Projet de Mesure de Température et d'Humidité

Le projet "Effet de Serre" est une application IoT développée pour mesurer et surveiller les conditions environnementales, en particulier la température et l'humidité. Ce projet utilise un capteur DHT11 connecté à une carte ESP8266 pour recueillir les données. Ces données sont ensuite mises à disposition via un point d'accès Wi-Fi, permettant aux utilisateurs de consulter les mesures en temps réel et de télécharger les données sous forme de fichier CSV.

Objectifs :

  • Surveillance des Conditions Environnementales
  • Accessibilité en Temps Réel
  • Téléchargement des Données

Technologies Utilisées :

  • ESP8266
  • DHT11
  • MicroPython
  • HTML/CSS
  • Socket Programming

Détails Techniques :

  • Initialisation et Configuration du capteur DHT11 et de l'ESP8266.
  • Mesure des Données à intervalles réguliers.
  • Interface Web responsive pour afficher les données actuelles et configurer les paramètres de mesure.
  • Téléchargement des Données en fichier CSV.

Utilisation :

  1. Accès au Point d'Accès : Connectez-vous au réseau Wi-Fi nommé "clebasmeteo" avec le mot de passe "clebas".
  2. Accès à l'Interface Web : Ouvrez un navigateur web et entrez l'adresse IP `192.168.4.1` pour accéder à l'interface utilisateur.
  3. Configuration des Mesures : Utilisez l'interface pour définir le nombre de mesures à prendre et l'intervalle entre chaque mesure.
  4. Téléchargement des Données : Après avoir pris les mesures, téléchargez les données en cliquant sur le bouton "Télécharger CSV".
